## Settings
This entries should be changed in order for bot to work:
- `https://api.telegram.org/bot{Token}` - Telegram's API token for bot [?](https://core.telegram.org/bots/api)
- `/feed?fields=id&limit=10&access_token={Token}` - FB Api token [?](https://developers.facebook.com/docs/facebook-login/access-tokens)
- `https://www.googleapis.com/youtube/v3/search?key{Token}` - YouTube API Token [?](https://developers.google.com/youtube/2.0/developers_guide_protocol_oauth2)

For DB connection:

```
	$dblocation = ""; // DB server location (usually localhost)
	$dbuser = "";	// DB ssername
	$dbpasswd = ""; // DB user password
	$dbname = "ttb"; // DB table name
```

Script must be handled through CRONJOB with access of every 1 minute (or 30 secs, as it is now on live server).
